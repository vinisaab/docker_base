# O que é um CONTAINER?

## Container é uma VM?
    - Não, container é mais sobre isolamento.
        - Isolamento de processos, usuários, network (Lógico)
            - Namespaces
        - Isolamento de recursos, CPU, IO (Físico)
            - CGroup

## Soluções
    - CHRoot
    - Jails (BSD)
    - Solaris
    - Open VZ
    - LXC (Linux Container)
    - Docker
